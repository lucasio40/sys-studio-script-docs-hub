## About

SYS-Studio is a FiveM-centered script hub created to present resources in a polished, community-friendly layout. It fits roleplay server teams, creators, and project owners who need one place to publish RP assets, keep references organized, and provide access to downloads and sales pages.

Instead of acting like a flat file list, the site is arranged for guided browsing. It includes curated script pages, documentation sections, and resource reference areas so visitors can move through showcase content and distribution details with less effort.

## What It Includes

- Script showcase pages for featuring selected FiveM resources
- Download-focused resource listings for shared RP assets
- Sales-ready resource pages for paid script entries
- Documentation and reference sections for setup notes and usage context
- Community-facing pages for creator portfolios and server branding
- Organized hub content for roleplay server teams and collaborators
- Website sections for browsing, presenting, and managing resources
- Support content for ongoing updates and content refreshes
